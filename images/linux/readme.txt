
root_default.img
	- 缺省安装的文件系统映象文件, 采用基于arm-linux-gcc-3.4.1的函数库
root_mizi.img
	- mizi公司提供的映象文件，含有中文手写识别，浏览器等。
root_qtopia_mouse.img
	- 标准的qtopia，使用鼠标操作,采用基于arm-linux-gcc-3.4.1的函数库
root_qtopia_tp.img
	- 标准的qtopia, 使用触摸屏操作。采用基于arm-linux-gcc-3.4.1的函数库
zImage_n35
	- 支持NEC 3.5" LCD的内核文件
zImage_a70
	- 支持7" LCD的内核文件
zImage_vga1024x768
	- 支持VGA扩展模块(分辨率1024x768@70Hz)的内核文件
