
本文件夹中的WinCE映像文件可以实现
- 注册表保存
- 实时时钟
- 支持三个串口驱动
- 支持.NET Compact Framework 2.0
- 支持SQLCE2.0
- 支持USB无线网卡(型号:vnuwlc5)
- 支持屏幕旋转
- 支持录音
- 支持中文拼音输入
- FTP服务
- TELNET服务
- WEB SERVER

请根据文件的名字使用各个文件

NK_*.bin - 代表windows ce映像文件
NK_*.nb0 - 代表运行时映像文件，可以借助supervivi的Download & Run功能通过USB下载到0x30200000运行

N35 - 表示支持NEC3.5寸LCD, 带触摸驱动
A70 - 表示支持7寸LCD, 带触摸驱动
